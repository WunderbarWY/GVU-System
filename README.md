# 银河先遣队作战指挥台

一个把 Linear 任务映射成太阳系战役地图的任务管理原型。

## 当前设定

- 银河先遣队：玩家势力，蓝色与冷白，控制地球和水星周围航道。
- 地球联合政府：商业活动战线，青绿色，控制月球和金星周围航道。
- 木星兵团：创作线活动，黄色调，控制木星和土星航道。
- 星际遗民：混乱势力，红色，盘踞冥王星和无人航道。

## 运行

直接打开 `index.html`，或在这个目录启动一个本地静态服务。

## 后续接入 Linear 的方向

1. 用 Linear API 拉取 issue、project、cycle 和 label。
2. 按 label 或 project 映射到商业、创作、杂事战线。
3. 用 priority、due date、status 和更新时间计算战斗力、补给、士气。
4. 在地图操作中回写 Linear 状态，例如开始推进、完成行动、拆分子任务。
